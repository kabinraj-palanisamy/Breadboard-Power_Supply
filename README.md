# Breadboard Power Supply
A multi-output regulated bench power supply PCB designed in KiCad 10,
intended for prototyping and breadboard use.

## Features
- Dual selectable output rails: fixed 5V (LM7805) and adjustable
  voltage (LM317)
- Output voltage options: 3.3V, 5V, 12V via onboard selector headers
- Barrel jack DC input with slide switch power control
- LED power indicator
- Dual output interfaces: breadboard pin headers + screw terminals

## Tools
KiCad 10.0.1
